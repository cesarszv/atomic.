# Atomic Web (Frontend)
Interfaz de usuario de **Atomic**, desarrollada con 
- **Next.js 15**
- **TypeScript** 
- **Tailwind CSS v4**. 
Proporciona una experiencia fluida y moderna para que los usuarios gestionen sus hábitos.

## Propósito
Ofrecer una plataforma interactiva para el registro de hábitos, visualización de analytics (heatmaps y gráficos) y gestión de configuraciones de cuenta.

## Estructura de Carpetas
- `src/app`: Rutas del App Router (Auth y Dashboard).
- `src/features`: Módulos encapsulados por dominio (habits, streaks, tracking, etc.).
- `src/shared`: Componentes UI reutilizables (estilo shadcn), hooks globales y cliente de API.
- `src/styles`: Configuraciones de Tailwind CSS y estilos globales.
